== Description ==
Image Hover Effects Pack is a collection of 150+ animated hover effects that are pure CSS3 based.It will allow you to add animated captions on the images with alot of customization options.It is 100% responsive and contains griding options.
It is the fastest and most easiest plugin to set up in just few minutes.
